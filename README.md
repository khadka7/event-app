## Installation

Clone this repo.

Rename `.env.example` to `.env` and fill the options.

Run the following commands:

```
composer install
npm install
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
npm run watch || npm run dev
```


## Note

You can run test with artisan command also. `php artisan test`
